 ### getaccept_hangman_game 
 
Hangman is an online guessing game for one player. 
The player should know the secret word and try to guess it by suggesting letters within a certain number of guesses.
  
 ### Dependencies

  - Node: [http://nodejs.org/](http://nodejs.org/)
  - ReactJS: [https://reactjs.org](https://reactjs.org)
  - Bootstrap: [https://getbootstrap.com](https://getbootstrap.com)

### Initial Set Up

1. You can run the code by using Visual Studio Code
2. Open the Terminal
3. Write`npm install`
4. `npm start` and it will start automaticaly or you can run the app port number `localhost:3000`
